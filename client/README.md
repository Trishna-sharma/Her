# Her by Mou

A Vite + React starter for a Bangladeshi shopping app.

## Getting started

1. Open a terminal in `client/`
2. Run `npm install`
3. Run `npm run dev`

## Project structure

- `index.html` - Vite entry point
- `src/main.jsx` - React bootstrap
- `src/App.jsx` - App shell and category skeleton
- `src/components/WelcomeScreen.jsx` - Landing welcome screen
- `src/styles.css` - Global styles
